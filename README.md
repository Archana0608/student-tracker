- Responsive and modern layout
- Background images for login/registration pages
- Clean color palette for all other pages

## Tech Stack

- **Backend:** Java, Spring Boot, Hibernate
- **Frontend:** HTML, CSS, Bootstrap (Thymeleaf)
- **Database:** MySQL
- **Build Tool:** Maven
- **IDE:** VS Code / Eclipse
